## 运行

- Win10 + MinGW

```sh
git clone https://github.com/miawua/MRTracer2d.git
cd MRTracer2d
mkdir build
cd build
cmake -G "MinGW Makefiles" ..
mingw32-make && raytracer2d.exe && start out.ppm
```
